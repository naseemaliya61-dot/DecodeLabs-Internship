# 🎣 Phishing Awareness Analysis

**DecodeLabs Industrial Training | Batch 2026**

## 📌 Overview

This project is part of the **DecodeLabs Cyber Security Industrial Training** program and focuses on **phishing awareness and threat detection**. It demonstrates how to identify phishing attempts by analyzing suspicious emails, recognizing common red flags, and following a structured process to safely handle potentially malicious messages.

The project also includes a **non-expert triage checklist** and a **decision tree** to help users make informed decisions when dealing with suspicious emails.

---

## 🎯 Objectives

- Analyze different types of phishing attacks.
- Identify suspicious email characteristics.
- Understand common social engineering techniques.
- Learn a simple incident triage process.
- Improve cybersecurity awareness and safe online practices.

---

## 🛡️ Types of Phishing Covered

- Mass Phishing
- Spear Phishing
- Whaling / Business Email Compromise (BEC)

---

## 🚩 Common Phishing Red Flags

- Suspicious or misspelled domains
- Generic greetings
- Urgent or threatening language
- Requests for passwords or financial information
- Sender and domain mismatch
- Suspicious links or attachments
- Requests to bypass company procedures
- Secrecy or confidentiality demands

---

## ✅ Non-Expert Triage Checklist

- Verify the sender's email address.
- Check if the domain is genuine.
- Look for urgency or fear tactics.
- Inspect links before clicking.
- Watch for suspicious attachments.
- Verify requests using a separate communication channel.
- Report suspicious emails instead of interacting with them.

---

## 🌳 Decision Tree

```text
Incoming Suspicious Message
               │
     ┌─────────┼─────────┐
     │         │         │
   SAFE   SUSPICIOUS  MALICIOUS
     │         │         │
     ▼         ▼         ▼
  CLOSE   VERIFY USER   BLOCK &
            REPORT      ESCALATE
```

---

## 🔑 Key Takeaway

> **Pause → Verify (through a trusted channel) → Report**
